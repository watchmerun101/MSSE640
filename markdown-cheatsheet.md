# Markdown Cheat Sheet

## Headings
- `# H1`
- `## H2`
- `### H3`
- `#### H4`
- `##### H5`
- `###### H6`

## Emphasis
- Italic: `*italic*` or `_italic_`
- Bold: `**bold**` or `__bold__`
- Bold Italic: `***bold italic***`
- Strikethrough: `~~strikethrough~~`

## Lists
- Unordered:
  - `- item`
  - `* item`
  - `+ item`
- Ordered:
  1. `1. first`
  2. `2. second`

## Links
- Inline: `[link text](https://example.com)`
- Reference:
  - `[example][1]`
  - `[1]: https://example.com`

## Images
- Inline: `![alt text](https://via.placeholder.com/150)`
- Reference:
  - `![alt][img1]`
  - `[img1]: https://via.placeholder.com/150`

## Code
- Inline code: `` `code` ``
- Code block (fenced):
  ```
  ```js
  console.log("Hello")
  ```
  ```
- Indented code block (4 spaces):
    code line

## Blockquotes
- `> Quote`
- Nested:
  > `> Nested quote`

## Horizontal Line
- `---`, `***`, or `___`

## Tables
- Basic:
  | Header 1 | Header 2 |
  |---|---|
  | Cell 1 | Cell 2 |
- Alignment:
  | Left | Center | Right |
  |:---|:---:|---:|
  | L | C | R |

## Task Lists
- `- [ ]` unchecked
- `- [x]` checked

## Misc
- HTML allowed in Markdown
- Line breaks: end line with two spaces  
- Escaping: use backslash `\` before special chars
